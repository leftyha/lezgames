# LEZGAMEZ

Monorepo base for the LEZGAMEZ platform: Next.js web, NestJS API, shared catalog, UI tokens/components and Game SDK contract.

## Commands

- `pnpm dev` runs apps through Turborepo.
- `pnpm build` builds the monorepo.
- `pnpm typecheck` validates TypeScript.

## MVP scope covered

- SEO routes, game detail pages and `/play/:slug` shell.
- Dark premium reusable visual system using the master tokens.
- Catalog model with 10 live games and 4 beta/coming soon entries.
- Wallet/store disclaimers: Lez Coins are internal credits only.
- API scaffold with mandatory module contracts.
